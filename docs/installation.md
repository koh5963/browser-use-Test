# Installation

## Stable release

To install Browser-Use-Test, run this command in your terminal:

```sh
uv add TestApp
```

Or if you prefer to use `pip`:

```sh
pip install TestApp
```

## From source

The source files for Browser-Use-Test can be downloaded from the [Github repo](https://github.com/koh5963/TestApp).

You can either clone the public repository:

```sh
git clone git://github.com/koh5963/TestApp
```

Or download the [tarball](https://github.com/koh5963/TestApp/tarball/master):

```sh
curl -OJL https://github.com/koh5963/TestApp/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd TestApp
uv pip install .
```
